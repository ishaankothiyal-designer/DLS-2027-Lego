# Future Expressive Palette

This file reserves space for a future non-brand visual gamut.

## Current Status

Not part of the active design-system token contract yet.

## Purpose

This future layer may support:
- marketing visuals
- campaign variation
- richer visual identity work that goes beyond the current brand and theme construct
- future brands that require more expressive range than the current system provides

## Important Boundary

The current exported `Misc` collection is not this expressive palette.

Today, `Misc` means:
- gap
- size
- stroke
- opacity

This future expressive palette is a separate concept and should be modeled separately if and when it becomes real.

## Promotion Criteria

Promote this concept into active token architecture only when:
- the palette is formally defined
- usage rules are documented
- dependency rules are clear
- Storybook and component implications are understood

## LLM Readiness

Load this file only when a task explicitly asks for:
- non-brand creative color systems
- marketing color exploration
- future palette expansion beyond the current design-system contract
