- [../../memory/workflows/figma-github-storybook.md](../../memory/workflows/figma-github-storybook.md)

## Use this skill for

- Figma handoff review
- token synchronization planning
- PR checklist design
- governance decisions

## Workflow

1. Treat Figma as design truth and GitHub as engineering truth.
2. Keep synchronization manual and reviewable.
3. Check alias direction, brand integrity, and Storybook impact in every meaningful change.

## Guardrails

- Do not assume automated sync.
- Do not accept token changes without checking downstream impact.
